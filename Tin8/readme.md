# Prosty serwer http

Autor: Kamil Wojas - s23878
Tytuł: Zadanie nr 8 

## Opis

Aplikacja serwera HTTP zbudowana przy użyciu frameworka Express, która implementuje szablony EJS. Trwałość danych jest zapewniona przez zapisywanie ich w pliku JSON lub w bazie danych MySQL, w zależności od ustawień środowiskowych. Aplikacja pozwala na wyświetlanie wszystkich rekordów encji produkty, dodawanie nowych rekordów oraz wyświetlanie szczegółów pojedynczego rekordu. Struktura aplikacji jest zorganizowana zgodnie z wzorcem MVC.
